=== Transformations
==== Dim_isEndgame
- 'Dim_isEngame' is a boolean value

==== Dim_GrandCompany
- 'Dim_GrandCompany' consists out of 3 Grand Companies and the lack of one.

==== Dim_Realms
- 'Dim_Realms' concats all the 'Category' of American_Realms, European_Realms, Japanese_Realms and Oceanian_Realms.
- 'Category' is renamed to 'Dim_Realms'

==== Dim_Regions
- 'Dim_Realms' concats all the 'Category' of American_Realms, European_Realms, Japanese_Realms and Oceanian_Realms.
- 'Category' is renamed to 'Dim_Realms'
- Adds 'V_Region' which contains their Server Locations / Intended Regions.

==== Dim_Tribes
- 'Category' is renamed to 'Dim_Tribes'.
- 'Tribe' is renamed to 'V_Interactions'.

==== Sub_Tribe_Categories
- 'Dim_Tribes' is copied from Dim_Tribes.
- Every Dim_Tribe gets a Furry category assigned, which was evaluated by professional furries. Said category is saved in 'V_Category'.
